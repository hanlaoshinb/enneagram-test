# 九型人格测试网站

这是一个使用HTML和TailwindCSS构建的九型人格测试网站。用户可以通过回答36个问题来了解自己的九型人格类型。

## 功能特点

- 36道精心设计的二选一测试题
- 详细的九型人格分析结果
- 包含典型人物、气质形象、恋爱特征等多维度分析
- 响应式设计，支持各种设备
- 现代化UI设计

## 技术栈

- HTML5
- TailwindCSS
- JavaScript (原生)

## 如何运行

1. 克隆本仓库
2. 在项目根目录打开 index.html
3. 开始使用！
